# Le Conjugueur — French Conjugation Trainer

A Parisian-chic, single-file French verb conjugation trainer.

## Features

- **9 tenses**: Présent, Passé Composé, Imparfait, Plus-que-parfait, Futur, Futur Antérieur, Conditionnel, Conditionnel Passé, Subjonctif
- **9 verbs**: parler, finir, être, avoir, aller, faire, savoir, écrire, lire
- **Smart pronoun selection** — each round pairs one singular + one plural pronoun from different grammatical persons
- **Mixed mode** — 4 verbs × 4 pronouns, each in a different randomized tense
- **Forgiving accents** — missing diacritics are accepted with a soft hint
- **Gender alternates** — `je suis allé` / `je suis allée` both accepted for *aller*
- **Streak tracking** + accent helper bar

## Run locally

Just open `index.html` in a browser. No build, no dependencies.

## Deploy

Static site — deploys anywhere. Currently hosted on Netlify.
